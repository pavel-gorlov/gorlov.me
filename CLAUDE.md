# Инструкции для работы с проектом gorlov.me

Этот документ содержит важную информацию для AI-ассистентов и разработчиков, работающих с проектом.

## Обзор проекта

**gorlov.me** - это персональный веб-сайт/портфолио, построенный на Astro с поддержкой мультиязычности и темной темы. Проект фокусируется на производительности, минимализме и доступности.

## Технологический стек

### Основные технологии

1. **Astro 5.8.2**
   - Статический генератор сайтов
   - Нулевой JavaScript по умолчанию
   - Компонентная архитектура
   - Файловый роутинг

2. **Tailwind CSS 4.1.8**
   - Utility-first подход
   - Компилируется через Vite плагин
   - Поддержка темной темы через классы

3. **i18next**
   - Интернационализация (RU/EN)
   - Файловая структура переводов
   - Динамическое переключение языков

4. **TypeScript**
   - Строгий режим включен
   - Типизация для Astro компонентов

5. **Sharp**
   - Оптимизация изображений
   - Автоматическое преобразование в WebP
   - Изменение размеров на лету

6. **pnpm**
   - Эффективное управление зависимостями
   - Поддержка workspaces

### JavaScript функциональность

#### theme_switcher.js
```javascript
// Основные функции:
- getTheme()        // Получить текущую тему
- applyTheme(theme) // Применить тему
- toggleTheme()     // Переключить тему
- updateIcons()     // Обновить иконки

// Особенности:
- localStorage для сохранения выбора
- Поддержка prefers-color-scheme
- Плавные переходы
- Inline-скрипт в <head> предотвращает мерцание (FOUC)
```

#### Предотвращение мерцания темы (FOUC)
В `Layout.astro` используется inline-скрипт в `<head>`, который применяет тему **до** отрисовки страницы:
```javascript
<script is:inline>
  (function() {
    const storedTheme = localStorage.getItem('theme');
    const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
    const theme = storedTheme || (prefersDark ? 'dark' : 'light');

    if (theme === 'dark') {
      document.documentElement.classList.add('dark');
    } else {
      document.documentElement.classList.remove('dark');
    }
  })();
</script>
```

**Важно**: Скрипт должен быть inline (`is:inline`) и находиться в `<head>` перед всеми стилями для мгновенного применения темы.
```

## Рабочие процессы

### Добавление новой страницы

1. Создать файл в `src/pages/новая-страница.astro`
2. Создать русскую версию в `src/pages/ru/новая-страница.astro`
3. Добавить переводы в `public/locales/*/translation.json`
4. Обновить навигацию в `Layout.astro`:
   ```javascript
   const newPagePath = `${i18next.language === 'ru' ? '/ru' : ''}/новая-страница`;
   ```
5. Добавить пункт меню в навигацию

### Изменение контента

1. **Тексты**: Редактировать JSON файлы в `public/locales/`
2. **Структура**: Изменять `.astro` файлы в `src/pages/`
3. **Стили**: Использовать Tailwind классы или `global.css`

### Добавление компонента

1. Создать файл в `src/components/НовыйКомпонент.astro`
2. Импортировать в нужный макет/страницу:
   ```javascript
   import НовыйКомпонент from '../components/НовыйКомпонент.astro';
   ```
3. Использовать в шаблоне:
   ```html
   <НовыйКомпонент />
   ```

## Отладка

### Частые проблемы

1. **Ошибка при сборке**
   - Проверить версию Node.js (>= 18)
   - Удалить `node_modules` и `pnpm-lock.yaml`
   - Переустановить: `pnpm install`

2. **Стили не применяются**
   - Проверить правильность Tailwind классов
   - Убедиться что файл включен в `tailwind.config.js`

3. **Переводы не работают**
   - Проверить пути в `public/locales/`
   - Проверить ключи в JSON файлах
   - Убедиться что `i18next.changeLanguage()` вызван
